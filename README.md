##  Overview

This project showcases an end-to-end Commercial Freight Operations Analytics solution developed using SQL Server and Power BI.

To overcome the lack of publicly available freight operations data, a realistic logistics database was designed and implemented in SQL Server, simulating shipment movements, transit performance, delivery status, route operations, and fleet activity.

The data was then transformed into an interactive Power BI dashboard to monitor logistics performance, identify operational bottlenecks, evaluate route efficiency, and support data-driven decision-making.

---

##  Business Objectives

- Monitor shipment performance across logistics routes
- Track transit delays and operational bottlenecks
- Analyze delivery efficiency and route reliability
- Evaluate fleet utilization and performance
- Improve visibility into freight operations
- Support strategic logistics planning
